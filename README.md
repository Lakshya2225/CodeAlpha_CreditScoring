# Credit Scoring Model

## Overview
This project predicts whether a customer has good or bad credit risk using Machine Learning classification algorithms.

## Dataset
- German Credit Dataset
- 1000 customer records
- Binary Classification (Good / Bad Credit)

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Joblib

## Workflow
1. Data Loading
2. Data Cleaning
3. Missing Value Handling
4. Label Encoding
5. Train-Test Split
6. Model Training
7. Model Evaluation
8. Model Saving

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Results

| Model | Accuracy |
|--------|----------|
| Logistic Regression | 66.5% |
| Decision Tree | 60.5% |
| Random Forest | **72.0%** |

## Best Model
Random Forest Classifier
